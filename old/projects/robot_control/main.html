Robot Control

Robot control is a personal project that aims to produce a general API using ROS2 for controlling any vehicles across many different simulators.

Supported simulators consist of AirSim, Ignition Gazebo, and Gazebo. There are plans to support pybullet in the future.

Since simulators can be interfaced with many different APIs I wanted to encapsulate the most common methods. Below is a table showing the simulator and the APIs that are supported. Native API means directly interfacing using that simulators API. For example, AirSim offers a native API and control through PX4.
